## Tips for Best Results on Luma

1. **Upload reference images**: Use the snapshots from `snapshots/` folder as "Image to Video" references — Luma generates much better when it has a visual starting point
2. **Start with Option 3** (individual clips) → stitch them together in CapCut or DaVinci Resolve → add your own music + text overlays
3. **For Option 1**: Luma currently works best at 5-10 second clips. Generate each scene separately, then edit together
4. **Resolution**: Luma outputs at 1080p. For 4K, upscale with Topaz Video AI
5. **Music**: Add music post-generation using [Suno AI](https://suno.ai) or royalty-free from [Uppbeat](https://uppbeat.io)
6. **Text overlays**: Luma struggles with exact text. Add text overlays in post-production (CapCut works great for this)

---

## Post-Production Workflow

```
1. Generate clips in Luma (Option 3 — individual features)
2. Import all clips into CapCut (free) or DaVinci Resolve
3. Arrange in sequence (follow the scene order from Option 1)
4. Add text overlays with exact wording
5. Add background music (Suno AI or royalty-free)
6. Add DialKaro logo as intro/outro
7. Export at 1080p, 30fps
8. Upload to:
   - Loom (as product demo)
   - LinkedIn (as native video)
   - YouTube (for SEO)
   - WhatsApp Status
```
