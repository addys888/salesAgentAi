# DialKaro — Ideal Customer Profile & Target List

> **Purpose**: Know exactly who to sell to and where to find them.
> **Rule**: Never spray-and-pray. Pick ONE segment per outreach session.

---

## The Perfect DialKaro Customer

| Trait | Why It Matters |
|-------|---------------|
| 5-50 sales reps | Too small for Salesforce, too big for WhatsApp groups |
| 50-500 inbound leads/day | Manual routing breaks above 30/day |
| Reps copy-paste from Excel | They feel the pain daily |
| Tier 1-2 Indian cities | Tech-aware, priced right |
| Owner is the decision maker | No procurement committee |
| Already uses WhatsApp | DialKaro's WhatsApp integration = "aha moment" |

## Segment 1: DSA Networks (Loan Agents) ⭐ HIGHEST PRIORITY

| Detail | Value |
|--------|-------|
| Team size | 5-50 reps per franchise |
| Products | Home loans, personal loans, credit cards |
| Lead sources | IndiaMART, JustDial, Meta Ads, referrals |
| Current tools | Excel + WhatsApp groups |
| Decision maker | Franchise owner / Principal Agent |
| Budget | ₹5K-25K/month |

### Where to Find

| Source | Search Term |
|--------|------------|
| IndiaMART | "Loan agents", "DSA franchise", "Home loan services" |
| JustDial | "Loan agents in [city]", "Personal loan agents" |
| LinkedIn | "DSA" OR "Direct Selling Agent" + India |
| DSA Federation of India | Member directory (public) |

### Cities: Mumbai → Delhi NCR → Pune → Hyderabad → Bangalore → Chennai → Ahmedabad → Jaipur

## Disqualification — Don't Waste Time On

| Red Flag | Why |
|----------|-----|
| Solo agent (1 person) | No team management value |
| Already on Salesforce/HubSpot | Switching cost too high |
| "We don't use WhatsApp" | Core advantage irrelevant |
| Enterprise (500+ reps) | Needs enterprise sales motion |
| No inbound leads | No routing problem |
