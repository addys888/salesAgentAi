# DialKaro — LinkedIn Showcase Page Setup Pack

A LinkedIn **Showcase Page** is a sub-page of your parent CelerApps company
page. Followers of the CelerApps parent see DialKaro updates only if they
explicitly follow the showcase. This gives DialKaro its own analytics,
followers, and posting voice — without splitting your main brand.

---

## 1. Prerequisite

The CelerApps **parent** page must already be live at
`linkedin.com/company/celerapps`. Showcase pages cannot exist on their own.

---

## 2. Create the Showcase Page

1. Go to `linkedin.com/company/celerapps/admin/`
2. In the top-right "Admin tools" dropdown → **Create a Showcase Page**
3. Fill in:

| Field | Value |
|-------|-------|
| Showcase page name | **DialKaro** |
| LinkedIn public URL | **linkedin.com/showcase/dialkaro** |
| Website | **https://dialkaro.celerapps.com** |
| Industry | **Software Development** |
| Company size | (inherited from parent) |
